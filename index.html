  1. HTTP Method 종류를 아는대로 설명해주세요.

    GET : 클라이언트에서 서버에 데이터를 요청할 때 사용한다.
    POST : 서버에서 요청 데이터를 클라이언트에 전송할 때 사용한다.
    PUT : 리소스를 완전 대체하는 메서드이다.
    PATCH : 리소스의 일부 수정하는 메서드이다.
    DELETE : 리소스를 제거하는 메서드이다.

  2. 인터넷과 웹의 차이에 대해 설명해주세요.

    인터넷은 통신 프로토콜을 이용해 정보를 주고받는 거대한 컴퓨터 통신망이다.
    웹은 인터넷에 연결된 컴퓨터를 통해 사람들이 정보를 공유할 수 있는 전 세계적인 정보 공유 시스템이다.
    인터넷을 구성하는 기술 중 하나가 웹이다.

  3. www.naver.com을 치면 무슨 일이 일어날까요?

    www.naver.com의 DNS를 조회한다.
    조회된 서버에 클라이언트는 HTTP 요청 메시지를 보낸다.
    요청 메시지를 받은 서버는 클라이언트에게 HTTP 응답 메시지를 보낸다.
