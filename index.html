Web 퀴즈!

  1. HTTP Method 종류를 아는대로 설명해주세요.

    GET: 리소스를 조회하는 메서드입니다.

    POST: 새로운 리소스를 생성하는 메서드입니다.

    PUT: 리소스를 완전히 대체하는 메서드입니다.

    PATCH: 리소스를 부분 변경하는 메서드입니다.

    DELETE: 리소스를 제거하는 메서드입니다.

  2. 인터넷과 웹의 차이에 대해 설명해주세요.

    인터넷은 전 세계 컴퓨터들을 하나로 연결하는 거대한 컴퓨터 통신망을 의미합니다.
    TCP/IP 프로토콜을 통해 제공됩니다.

    웹은 하이퍼텍스트 문서로 이루어진 웹 페이지들이 링크로 연결되어있어 서로 이동할 수 있는 정보 공간을 의미합니다.
    HTTP 프로토콜을 통해 웹문서에 누구나 검색하고 접근할 수 있습니다.
    웹은 인터넷을 구성하는 여러가지 기술들 중 하나입니다.

  3. www.naver.com을 치면 무슨 일이 일어날까요?

    1. 브라우저는 해당 주소로 HTTP GET 요청을 보냅니다.
    2. 네이버 서버는 이 요청을 받아들이고, 네이버 웹 페이지의 내용과 함께 HTTP 응답을 보냅니다.
    3. 브라우저는 이 응답을 받아들여서, 네이버의 웹 페이지를 표시합니다.
